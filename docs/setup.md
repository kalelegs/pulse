# Setup

0. Install bun

```bash
curl -fsSL https://bun.sh/install | bash
```

1. Install dependencies

```bash
bun install
```

2. Configure environment variables in `.env`

```bash
OPENAI_API_KEY=your_openai_api_key
# todo: add remaining
```

3. Start the app

```bash
bun run dev
```

4. Open [http://localhost:3000](http://localhost:3000)
